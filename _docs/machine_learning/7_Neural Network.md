## Neural Networks

hidden layer가 한개 있는 neural network를 살펴보자.

![image-20201027150556250](https://i.loli.net/2020/10/27/856R3xQNei4AShV.png)

* 입력 units은 세개이고, activation function이 사용된다.
* 만약, hidden layer의 유닛 개수가 네개라면, matrix $\mathbf{w}$의 크기는 $[3, 4]$이 된다. 그리고 $\mathbf{x}$의 크기는 $[3,1]$이 된다. 
  * 추가적으로 bias $\mathbf{b}$의 크기는 $[4,1]$이다.
